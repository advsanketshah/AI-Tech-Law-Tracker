## Selection Criteria

Not every article from these sources makes it into the tracker. Items are selected based on:

1. **Significance** - Does this change the legal/regulatory landscape?
2. **Impact** - Does this affect how organizations deploy technology?
3. **Novelty** - Is this a new development or just commentary on existing news?
4. **Actionability** - Do practitioners need to know this for their work?
5. **Global relevance** - Does this set precedent beyond one jurisdiction?
